■ 概要

キーボード主体の検索型ランチャ
どこにあるファイルでも素早く開くことが出来るようになります。

■ 最初の起動

fenrir.exeを起動すると、ドライブをスキャンするかどうか問われます。
スキャンはファイル検索のためのデータベース作成の為に行われます。
ここで「はい」を選択するとスキャンが始まります。
しばらく待つとスキャンが終わり、使用できるようになります。

■ アーカイブの内容

fenrir.exe
docs\
data\

■ インストール

専用のフォルダをつくり、そこにアーカイブの中身を入れてください。

■ アンインストール

レジストリは変更していませんし、インストールフォルダの外部には
ファイルやフォルダを作成していません。
このソフトが入っているフォルダごと削除すればOKです。

■ 設定

メニューからOptionを選択すると設定ダイアログが開きます。
設定はここで行います。
見ればわかると判断した部分の説明は省きます。
分からないところがあったら質問してください。

・最大列挙数
候補リストに最大でどれだけアイテムを列挙するかを指定します。
少なければ少ないほど高速に動作します。

・ スキャンの設定

[スキャンの設定] をおすとdata\フォルダにあるscan.iniという
ファイルが開きます。
このファイルではスキャンする場所、登録するファイルの種類を管理しています。
このファイルを編集することで、スキャン登録の制御を行なうことが可能です。

一行目には取得するファイルのマスクを指定します。
マスクは複数指定できます。
各マスク間は セミコロン(;)一文字で区切ります。
フォルダを登録したい場合は、\ を指定します。

二行目以降に、スキャンしたい場所をフルパスで指定します。
複数指定したい場合は改行で区切ってください。
スキャンする場所以下に取得除外したい場所が在る場合は、
-C:\RECYCLED のように除外指定することも可能です。

例)C:\ F:\ G:\tools 以下の *.exe と *.ini と readme.* と フォルダを登録。
ただし C:\RECYCLED 以下は取得しない。

-----------------------------------------------------------
\;*.exe;*.ini;readme.*
C:\
F:\
G:\tools
-C:\RECYCLED
-----------------------------------------------------------

・ 順位の保存

実行したアイテムは、候補に優先的に表示されるようになります。
この優先順位を上位何件までスキャン後に持ち越すかを指定します。
ゼロにした場合は、スキャンを行うと全ての順位が破棄されます。
ここに大きな値を指定するほど、スキャン後の処理に時間を要します。

・ 使用中にリンク切れパスの自動削除を行う

リンク切れのチェックは候補を表示するタイミングで行います。
したがって、ここの設定は候補表示時のパフォーマンスに影響します。

■ 内部コマンド

入力ウィンドウに入力して使います。

/s	Scan
/o	Option
/a	About
/x	Exit
/?	Help

■ インスタントコマンド

任意の文字列に任意のコマンドを割り当てることができます。
data\フォルダにある_instant.iniをinstant.iniに改名すると
機能が有効になります。

予め例として三つほどコマンドを設定しています。
/nや/cなどと入力するとnotepadや電卓が起動するはずです。
/g 検索語と入力するとGoogleで検索語を検索します。

詳しくはinstant.iniの中身を参照してください。

■ ただの関連づけ実行以上のことがしたい場合

[モディファイアキー(Alt,Ctrl,Shift,Win)] + [A-Z,ENTER,SPACE,F1-F12,TAB,0-9,up,down,left,right,app]の
任意のキーの組み合わせに、選択したパスを絡めた任意のコマンドまたはメニューを
割り当てることができます。
docs\cmd.txtを参照してください。

■ 起動オプション

/t 			非常駐モードで起動。
/key=任意の文字列	= 以降の任意の文字列を入力。
/key= に指定する文字列中にスペースが含まれている場合は""で囲む必要があります。
例)fenrir.exe /t /key="c:\program files\*.ini"
/dir=任意の文字列	= 以降の指定ディレクトリを開く

以下のオプションを指定すると、使用するファイルやフォルダをデフォルトの物から変更することができます。

/scanfile=		スキャンファイル（デフォルト：data\scan.ini）
/pathfile=		インデックスファイル(デフォルト:data\path)
/initfile=		動作設定ファイル(デフォルト：data\fenrir.ini)
/cmddir=		コマンドファイルの場所。
